# Brain Cleaner

A professional utility to remove AI residue and "brain" logs from your system.

## Features
- Scans for `.gemini`, `.claude`, `.cline`, `.cursor`, and other AI agent folders.
- Modern Dark/Light mode UI using `CustomTkinter`.
- Selective cleaning or "Clean All" functionality.
- Real-time activity log.

## Requirements
- Python 3.7+
- `customtkinter` (Install via `pip install customtkinter`)

## How to Run
```bash
python3 app.py
```

## Disclaimer
This tool deletes folders permanently. Please review the lists of discovered residues before performing the cleaning.
